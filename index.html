Vaccine Availability INDIA coWIN
    Derived from the official Government of India Portal Co-WIN. Beneficial for the ones who are struggling to find a vaccination center with an available slot in your district.
    If you find an available slot, immediately book the slots from the official portal for Co-WIN Book Slot Registration.
PLEASE WEAR MASK ALL THE TIME WHEN OUTSIDE -- This website only shows available slots for vaccination does not book slot for vaccination


Features: Saving your state and district, removing unavailable slots, alert you with a GUITAR TONE on finding an available slot, the rest is same as the original Co-WIN.
Implementation: Created using the Co-WIN Public API This tool is open source, you can find the code here at my git [link].
Known bugs: The data might be inconsistent with the actual website, because the public API gives slightly older cached data.
